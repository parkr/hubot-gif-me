hubot-gif-me
============

Bring your personal gifs into your favourite chat client.

### Usage

```text
/gif me
/gif me itworks
```

### Installation

```bash
$ npm install --save hubot-gif-me
$ hk set HUBOT_GIF_INDEX="http://example.com/gif_index.json"
$ vim external-scripts.json # add "hubot-gif-me" to the array
```
